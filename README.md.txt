# Customer Churn Prediction System

## Overview
This project predicts customer churn using machine learning and provides insights through an interactive dashboard.

## Features
- Random Forest churn prediction
- RFM based segmentation
- SHAP explainability
- Streamlit web app

## Tech Stack
Python, Streamlit, Scikit-learn, SHAP

## Run Locally
pip install -r requirements.txt
streamlit run app.py
